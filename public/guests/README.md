# Фото для блока «Важные гости»

Положите портреты в эту папку (`public/guests/`) в формате **JPG**, **PNG** или **WebP**.

## Имена файлов (как в `data/guests.json`)

| Файл | Гость |
|------|--------|
| `dmitry.png` | Дмитрий Служеникин |
| `alexander.jpg` | Александр Болховитянов |
| `sergienko_sergej-1.jpg` | Сергей Сергиенко |

Можно использовать `.png` или `.webp` — тогда измените расширение в `data/guests.json` в поле `photoPath`.

## Рекомендации

- Квадрат или вертикальный портрет, минимум **400×400** px
- Лицо по центру кадра
- Нейтральный или однотонный фон

## Должность

Тексты ролей задаются в `data/guests.json` (поле `role`).
