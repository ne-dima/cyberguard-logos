# Фото для блока «О КиберСтраже»

Положите сюда изображения в формате **JPG**, **PNG** или **WebP**.

Рекомендуемые имена (как в `src/content/aboutSlides.ts`):

- `photo-1.jpg`
- `photo-2.jpg`
- `photo-3.jpg`
- `photo-4.jpg`, `photo-5.jpg` (и далее по номеру)

Можно добавить больше файлов: отредактируйте список `ABOUT_SLIDES` в `src/content/aboutSlides.ts`.

Рекомендуемый размер: от **1600×900** px, горизонтальная ориентация, важные детали не по краям (слева будет текст с градиентом).
